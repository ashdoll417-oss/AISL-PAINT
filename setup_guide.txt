Flask + Supabase Setup Commands (Windows):

1. Create & Activate Virtual Environment:
`
python -m venv venv
venv\Scripts\activate
`

2. Install Dependencies:
`
pip install flask==3.0.3 supabase==2.28.0 python-dotenv==1.0.1 gunicorn==23.0.0
`

3. Generate requirements.txt:
`
pip freeze > requirements.txt
`

4. Test Flask App:
`
python app.py
`

5. Deactivate (when done):
`
deactivate
`

### Render Deploy Files Ready:
